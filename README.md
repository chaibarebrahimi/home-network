# home-network
documentation of my first small home network
## Topology
- FritzBox 7590 as main router
- 2.4 GHz and 5 GHz WLAN enabled
- 1 PC via LAN
- 2 smartphones via WiFi

## DHCP
- DHCP enabled on router
- Standard gateway: 192.168.178.1

## Security
- WPA2/WPA3 enabled
- Guest network disabled
